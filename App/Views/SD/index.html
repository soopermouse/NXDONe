Service Desk Workflow

RMA Worlflow

  • Check the list of submitted RMAs On RMA Intake Page
  • Contact the customer if necessary or accept the RMA if the necessary conditions are met
  • If the RMA is accepted, send a shipping label to the customer
  • When the devices arrive book them in on the RMA Shipping Page
  • When the Device is repaired/ replaced, print the shipping label
  • Return the device or replacement and close the order

Rebuy Worlflow

  • Receive Devices
  • Update shipping status on Shipping Status page
  • Check the status of offers on the Rebuy Status page
  • If the offer is accepted send order for payment on the Payments Page
  • If the offer is not accepted return the device on the Returns page
  • Close Order

Customer Support

  • Edit customer data on the Customer Data Page
  • Edit order data on the Order Edit page
  • Edit payment data on the Payment Edit page
