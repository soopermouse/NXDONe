Service Desk Workflow

  • Receive and scan in device
  • Update shipping status on Shipping Status page
  • Check offers and send them to the customer from the Mail page
  • Check the status of offers on the Status page
  • If the offer is accepted, send it for payment on the Payment page
  • If the offer is rejected, send a second offer on Second Offer page
  • If second offer is accepted, send it for payment on the Payment page
  • If the second offer is rejected, return the device on Returns page
  • If the customer requested it, send the device to be recycled on Recycle page
  • Close the Order on Close Orders page

Technician Workflow

  • Check orders on the Check page and enter the details
  • If the device passes the check start Inspection on the Inspection page
  • Recycle the devices sent to Recycle on Do Recycle page

Manager Workflow

  • Check new orders on Latest Orders page
  • Check the status of orders on the Reports page
  • Set offers on Offer page
  • Send offers from Mail page
  • Set and send second offers from Second Offerpage
