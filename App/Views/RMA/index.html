RMA Workflow
  • Receive RMA Order
  • Check IMEI
  • If the device was bought from us and is in warranty, contact customer
  • If SD is satisfied with reason for RMA, issue a RMA token
  • Customer uses the RMA token to create a shipping label and send the device to us
  • Device is received and inspected
  • Device is sent to repair
  • If necessary device is sent to repair 2
  • If the device is repaired, issue a shiping label and send back
  • If the device cannot be repaired, send a replacement
  • close RMA order
